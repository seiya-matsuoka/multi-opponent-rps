指定した人数（1〜10）の相手（CPU）と、まとめてじゃんけんを行うミニアプリ。  
バックエンドは Spring Boot の REST API、フロントは React + TypeScript（Vite）で実装（別ホスティング）。

- バックエンド：Render（Dockerfile でデプロイ）
- フロントエンド：Vercel（Vite ビルド成果物をデプロイ）
- DB：使用なし（ステートレス）
